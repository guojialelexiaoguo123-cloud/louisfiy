# 欢迎来到我的个人小站，开篇碎碎念

大家好，欢迎访问我的个人博客网站。

这是我第一次自己从零搭建个人博客，从挑选域名、对比后缀、注册域名，再到接入 Cloudflare 解析、配置网站，前前后后折腾了好几天，终于正式上线了。

## 为什么要建这个博客
我只是一名普通高中生，不是什么技术大佬，也不会高深的编程知识。

单纯就是喜欢折腾、喜欢捣鼓这些小东西，想拥有一个**完全属于自己的网络小角落**。
不用受社交平台限制，不用迎合任何人的风格，想写什么就写什么，想记录什么就记录什么。

## 这里以后会更新什么
我不打算写那种很难、很硬核的技术文章，门槛太高自己看着都累。
以后这里大概率会分享这些：

- 搭建网站、域名、解析这类**超简单入门小教程**
- 自己折腾电脑、网络、小工具的踩坑记录
- 日常学习感悟、生活碎碎念
- 看到的好文、好句、有意思的分享
- 自己做的小练习、小项目记录

都是**技术含量不高，但很实用、很接地气**的内容，适合新手、和我一样喜欢瞎折腾的人看。

## 写给和我一样的新手
其实很多看起来很复杂的东西，只要愿意一步步跟着做，都能搞定。

一开始我也看不懂域名、解析、DNS 这些名词，看着一堆设置一头雾水，一边查一边试，慢慢就懂了。
所以不用觉得自己基础差、不敢尝试，**敢动手折腾，就已经赢过很多人了**。

往后就在这个 `.cyou` 的小站里，慢慢更新、慢慢成长。

感谢每一位路过的访客，我们来日方长。

—— 记于博客正式上线第一天
